# Healthcare Data Dashboard & SQL Analysis

## Overview
This project demonstrates healthcare data analysis using SQL and Power BI. It focuses on patient risk, clinic performance, and data quality.

## Dashboard Highlights
- Patient risk analysis (high, medium, low risk)
- Key risk drivers (ER visits, abnormal labs, active medications)
- Clinic revenue and utilization
- High-risk patient identification

## SQL Analysis
The SQL queries support:
- Patient risk indicators
- Revenue and financial breakdowns
- Operational trends (visits, providers, clinics)
- Data quality checks (duplicate patients, multiple systems)

## Tools Used
- SQL (MySQL)
- Power BI

## Notes
The dataset used in this project is synthetic and created for practice purposes.
